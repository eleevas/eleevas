Dental lab web app

Case Study.

Brief

During a job interview I was asked "What do you think an app for dental lab should have?". With my background as a dental lab technician, I was able to provide answers off the top of my head, but I kept exploring features and decided to create a passion project named DentOS.

I'm working on this project in my free time, so please bear with me.

My role

I'm working alone on this project.

I have to ideate the concept, featureset, branding, UX / UI, validate design decisions using UX research methods, create design system, and build hi-def prototypes.

As an optional goal, I might wrap it up with interactive HTML + CSS prototypes.

Feature

Device monitoring

Scanners, printers, mills can be monitored in real time, letting users know if they're online, offline, idle, or had an error

Users can:

  • Get an overview of all tasks and time to completion in realtime
  • Prioritize task queue per device
  • Schedule tasks for later
  • Remotely print
  • Be alerted via push notifications
Feature

Inventory tracking

Resins, solvents, plates, printer tank volume, wax, tools, etc. can be added to inventory. Inventory usage can be customized and tracked over time. Notification to restock would show up to the user.

Optional automated restocking process in collaboration with local suppliers and delivery partners ensure dental labs are never out of necessary materials for daily operation.

Feature

In-app store

Users can:

  • Purchase software licenses
  • Purchase materials in colaboration with local partners with arranged delivery
  • Connect to local dental polyclinics for taking orders (i.e. via 3Shape Unite)
  • Integrated payment gateway and ability to store payment methods for automated purchases
Feature

Multi user environment

  • Admin / user accounts with access control and device sharing
  • Activity log
  • 2FA authentication
Feature

Other

  • Automated cloud syncing provides means for at a glance overview of dental lab operations on any device, at any time
  • Patient cards with detailed history
  • Contact book
  • Materials, tasks, time-cost statistics
  • Calendar of events
  • Built with HIPAA / GDPR compliance in mind
